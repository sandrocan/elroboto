\section{Results and Discussion}

It should discuss:

\begin{itemize}
    \item successful embedded servo communication;
    \item physical validation of the kinematic solver;
    \item execution of the square waypoint sequence;
    \item controller behavior;
    \item benchmark conclusions;
    \item sensor validation and final integration status;
    \item limitations of the proof of concept.
\end{itemize}

Important limitations include:

\begin{itemize}
    \item four active kinematic joints rather than full six-DoF pose control;
    \item fixed wrist rotation and gripper state;
    \item no general trajectory planning;
    \item no guaranteed straight-line Cartesian interpolation;
    \item no systematic workspace or singularity characterization;
    \item no dynamic robot model;
    \item no exhaustive unreachable-target testing.
\end{itemize}
